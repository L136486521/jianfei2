体重跟踪减肥助手
一个专为减肥人士设计的体重跟踪应用，帮助您记录体重变化、管理减肥日记，并通过图表直观展示减肥进展。

功能特点
📊 体重记录
早晚记录：支持早晨和晚上分别记录体重

自动更新：记录后自动更新统计数据和图表

历史查看：查看最近7天的体重记录

📈 数据统计
初始体重：记录您开始减肥时的体重

最轻体重：显示减肥期间的最低体重

最重体重：显示减肥期间的最高体重

平均体重：计算所有记录的平均值

体重差值：显示体重波动范围

📉 趋势图表
多种视图：可选择早晨体重、晚上体重或全部体重

时间范围：支持查看最近7天、30天或全部数据

交互操作：支持缩放和滚动查看详细数据

直观展示：通过折线图清晰展示体重变化趋势

📝 减肥日记
饮食记录：记录每日饮食内容

心得分享：记录减肥感受和心得

自动保存：当天日记可随时修改和保存

历史回顾：查看最近10天的日记记录

💾 数据管理
数据导出：将数据导出为Excel文件，包含体重记录和减肥日记两个工作表

数据导入：从Excel文件导入数据

文件位置：查看导出文件的具体位置

数据备份：支持数据备份和恢复功能

安装与运行
环境要求
Python 3.7+

Kivy 2.0+

pandas

openpyxl

安装步骤
安装Python依赖包：

bash
pip install kivy pandas openpyxl
下载应用代码文件 main.py

运行应用：

bash
python main.py
Android设备运行（使用Pydroid）
在Pydroid中安装以下包：

kivy

pandas

openpyxl

将 main.py 导入Pydroid

直接运行即可

使用说明
首次使用
打开应用后，系统会自动创建数据库

在"记录体重"页面开始记录您的体重

建议每天早晚各记录一次，以获得准确数据

日常使用流程
记录体重：每天早晚在"记录体重"页面记录体重

查看统计：在"体重统计"页面查看减肥进展

分析趋势：在"趋势图表"页面查看体重变化趋势

写日记：在"减肥日记"页面记录饮食和心得

定期备份：在"数据管理"页面定期导出数据备份

数据备份建议
建议每周导出一次数据到Excel文件

将导出的Excel文件保存到安全位置

如需更换设备，可通过导入功能恢复数据

注意事项
体重记录
体重范围：20-400斤

建议在固定时间记录体重（如早晨起床后、晚上睡前）

使用同一台体重秤以获得准确数据

数据安全
应用数据存储在本地设备中

请定期备份重要数据

卸载应用前请确保已导出数据

使用建议
坚持记录：每天坚持记录体重和日记

定期回顾：每周查看一次趋势图表，了解减肥效果

合理目标：设定合理的减肥目标，健康减重

结合运动：记录饮食的同时，合理安排运动
